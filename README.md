# Sinatra App Template

A starting point for a lightweight app with:

* [Sinatra](http://www.sinatrarb.com)
* [Slim](http://slim-lang.com)
* [LESS](http://lesscss.org)
* [CoffeeScript](http://coffeescript.org)
* [Heroku](http://www.heroku.com)

## Usage

`ruby app.rb`
